Project 2: Digital Filling Station

  • Category: PLC Programming
  • Software: RSLogix Micro Starter Lite, RSLinx Classic Lite, RSLogix Emulate 500
  • Course: PLCDojo.com > Applied Logic (Level 2)
  • Project date: 07 October, 2024
  • Project PDF: Project 2 - Digital Filling Station.pdf

Pecan and Walnut Filling Machine

  • Machine that will fill empty boxes with either Pecans or Walnuts.
  • This machine will move empty boxes down a conveyor line until reaching a Proximity Sensor.
  • Afterwards, a Photo Eye reads either a Red or Blue barcode to determine type of Nut
  • Once a box is full, a Level sensor signals the PLC to continue to move the box down the line
Custom Additions:
  • System for keeping track of machine states:
      • S0: Emergency Stop
      S1: Initial Condition for Motor Start
      S2: Box Arrives at Filling Station
      S3: Box Filling with Pecans (Red Barcode)
      S4: Box Filling with Walnuts (Blue Barcode)
      S5: Box Filling Complete (Includes 5 second timer to run conveyor before returning to State 1)
  • Emergency Stop Button and Machine Reset Button
  • MODE STR: "Mode String" for easy checking on the machine system mode as a string
