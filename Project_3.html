Project 3: Inventory Management

  • Category: PLC Programming
  • Software: RSLogix Micro Starter Lite, RSLinx Classic Lite, RSLogix Emulate 500
  • Course: PLCDojo.com > Applied Logic (Level 2)
  • Project date: 09 October, 2024
  • Project pdf: Project 3 - Inventory Management.pdf

A Simple Inventory Management PLC Program

  • This program will take in a string (Simulating a Barcode Reader Input) and will parse string to locate the the Part Number, Quantity and Direction (Checking Parts In or Out of the system).
  • Currently tracks three active part numbers located in the Part Number String File.
  • System will not add part numbers that do not exist.
  • System simulated a PLC program performing a task that should be done using a traditional programming language.
Custom Additions:
  • System will provide alarms to the user if there is an attempt to overdraw an amount greater than the current Inventory.
  • System will also provide an alarm if the part number is not recognized upon scan.
  • If there are no errors, system will provide feedback to let the user know there was a good scan.
Opportunities for Improvement:
  • Addition of a way to save or export inventory data.
  • Reorganizing system to allow for additional part number to be added with ease instead of being hard coded.
  • Ability to check the character limit to make sure that the scanned part number is within specification.
